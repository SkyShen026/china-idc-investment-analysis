# 技术方案：中国数据中心投资量化模型

## 1. 项目目标

**一句话**：给定一组候选城市，输出每个城市新建 AI 数据中心的财务回报（IRR/NPV/Cap Rate）和综合选址评分，并可视化为全国机会热力图。

**回答的投资问题**：
- 哪些城市的新建 IDC 项目 IRR > 12%？
- "东数西算"政策节点的实际财务回报是否真的优于非节点？
- 电价每下降 0.05 元/kWh，IRR 提升多少？（敏感性）
- 给定 50 亿预算，最优投资组合是什么？

---

## 2. 投资模型框架

### 2.1 收入模型 (Revenue)

数据中心收入 = 机柜数 × 单机柜月租金 × 12 × 入住率

**参数**：
- 机柜数：通常 1000-5000 个/项目
- 单机柜月租金：北上广 ¥6000-8000 / 二线 ¥4000-5500 / 西部 ¥3000-4000
- 入住率：成熟项目 85-95% / 爬坡期 30-70%

### 2.2 成本模型 (Cost) — 电力是大头

**OPEX 拆分**（年度）：
| 项 | 占比 | 备注 |
|---|---|---|
| 电力成本 | **55-70%** | PUE × IT 负载 × 8760h × 电价 |
| 折旧 | 15-20% | CAPEX / 折旧年限 |
| 人力 | 5-10% | 运维团队 |
| 运维耗材 | 5% | 备件、化学品 |
| 其他 | 5% | 网络带宽、税费 |

**电力成本核心公式**：
```
年度电费 = 单机柜功率(kW) × 机柜数 × PUE × 8760 × 工商业电价(元/kWh)
```

**关键变量**：
- **PUE (Power Usage Effectiveness)**：总能耗/IT能耗。新建标准 1.15-1.3，老旧 1.5+。
- **单机柜功率**：传统云 4-6kW / AI 训练 10-30kW / 推理 6-10kW
- **工商业电价**：东部 ¥0.65-0.85 / 西部 ¥0.30-0.45（区别巨大！）

### 2.3 现金流 & IRR

```
年度自由现金流 = 收入 - OPEX - 税
项目 NPV = Σ FCF_t / (1+r)^t - CAPEX
项目 IRR = 使 NPV = 0 的折现率
Cap Rate = NOI / 资产价值
```

典型项目结构：
- 建设期：1-2 年（CAPEX 集中投入）
- 爬坡期：1-3 年（入住率从 30% 爬到 80%+）
- 稳定期：10-15 年
- 折旧：建筑 30 年 / 机电 10 年 / IT 设备不在 IDC 资产里

### 2.4 综合选址评分

不是所有项目都做财务回报最高的，还要看：

| 维度 | 权重 | 评分逻辑 |
|---|---|---|
| 电价 | 30% | 越低越好（核心成本） |
| 政策（东数西算节点） | 20% | 节点 +20，集群 +10 |
| 气候（年均温） | 15% | 越冷越好（PUE 低） |
| 距需求中心延迟 | 15% | 距 1000km 内核心城市群 |
| 电网容量 | 10% | 是否有富余装机 |
| 土地成本 | 5% | 工业用地基准价 |
| 政府补贴 | 5% | 税收返还、电价补贴 |

**综合分** = Σ 维度归一化分 × 权重，0-100 分

## 3. 数据架构

### 3.1 数据源分层

```
data/
├── reference/          ← 手工整理的稳定基准数据（开箱即用）
├── raw/                ← 通过脚本爬取/下载的原始数据
└── processed/          ← 清洗后的分析就绪数据
```

### 3.2 关键数据源

| 数据 | 来源 | 获取方式 | 更新频率 |
|---|---|---|---|
| 各省工商业电价 | 各省发改委文件 | PDF 人工/OCR | 半年 |
| 东数西算节点 | 国家发改委 | 固定 | 几乎不变 |
| 上市公司 IDC 位置 | 公司年报 | PDF 解析 / Wind | 年度 |
| 机柜租金 | 行业报告（科智咨询、IDC 圈） | 人工收集 | 季度 |
| 气候数据 | 中国气象局 / NOAA | API | 年度 |
| 工业用地价格 | 国土资源部 | 月度公告 | 月度 |

### 3.3 候选城市清单（25 个）

**Tier 1（核心城市群）**：北京、上海、广州、深圳
**Tier 2（核心枢纽周边）**：南京、杭州、苏州、武汉、成都、重庆
**东数西算 8 节点城市**：贵安新区、中卫、庆阳、和林格尔、芜湖、韶关、张家口、长三角生态绿色一体化示范区
**潜力新兴**：呼和浩特、银川、兰州、西宁、乌兰察布、惠州、清远

## 4. 模型实现

### 4.1 模块拆分

```python

├── data_collection/     # 数据采集（这里可以体现工程能力）
│   ├── electricity_prices.py   # 电价数据 ETL

│   ├── listed_company_idcs.py   # 上市公司年报 IDC 信息提取

├── model/               # 财务建模（CFA 知识展示）
│   ├── revenue.py
│   ├── cost.py
│   ├── irr.py
│   └── site_score.py
└── visualization/       # 可视化（DS 能力展示）
```
